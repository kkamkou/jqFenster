jqFenster

Lightweight popup framework

Examples (Inline)

Simple link
<a href="/some/thing" data-selector="#target" class="jqFenster">Simple link</a>
With animation
<a href="/some/thing" data-selector="#targetSecond" data-options="{animationSpeed: 400}" class="jqFenster">With animation</a>
Without overlay and animation
<a href="/some/thing" data-selector="#targetSecond" data-options="{animationSpeed: 0, noOverlay: true}" class="jqFenster">Without overlay and animation</a>
Callback
<a href="/some/thing" data-selector="#targetSecond" data-options='{callbackOpen: "myOpen", callbackClose: "myClose"}' class="jqFenster">Callback</a>
Remote one
<a href="remote.html" class="jqFenster">Remote one</a>
<button data-selector="#targetSecond" class="jqFenster">I'm button</button>
<button id="callbackOpenChanger">change the callbackOpen option globally</button>
